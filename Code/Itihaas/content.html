HTML

1. HTML is a markup language used to create static web pages and web applications.
2. Consists of tags surrounding content.
3. HTML cannot be used in a CSS file.
4. It is used to build the structure of the web pages.

CSS

1. CSS is a style sheet language responsible for the presentation of documents written in a markup language.
2. Consists of selectors succeeded by a declaration mark.
3. CSS can be used in an HTML file.
4. It is used to make web pages more presentable.
